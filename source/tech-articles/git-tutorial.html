如何使用 Git 及 GitHub 配置

1. 将本地 Git 的 SSH 密匙配置到 GitHub

首先需要下载 Git 软件。安装完成后，在菜单搜索 git bash 并且打开执行以下命令：

cd ~/.ssh
cat id_rsa.pub

如果显示没有密匙，就执行以下命令生成：

ssh-keygen -t rsa -b 4096

生成后，打开 GitHub 网页，进入 Settings -> SSH and GPG keys。新建一个 SSH key，然后把刚刚复制的密匙粘贴进去保存。

2. 将项目克隆到本地

新建一个空目录，然后右击鼠标选择 Git Bash Here 执行以下命令：

git clone [你的仓库地址]

3. 创建并管理自己的分支

使用你的编辑器（如 VS Code, Cursor, Windsurf等）打开项目目录，然后执行：

git checkout -b [分支名]   # 创建并切换到自己的本地分支
git push -u origin [分支名]  # 将自己的分支推送到远程

然后在 Cursor 或 Windsurf 的左下角点击，可以切换到自己的分支。

注意：不要点远程分支！每次写代码都在这个（你的本地）分支下写。

每天修改的代码，可以在完成之后，推送到自己的远程分支（注意：一定要在自己的分支推送）。
