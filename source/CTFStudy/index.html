📚 学习导航

🖥️ Web 安全 (Web Security)

Web 漏洞挖掘与利用，占据 CTF 比赛半壁江山的核心基础。

核心知识点：

  • SQL 注入 (SQLi) & 跨站脚本攻击 (XSS)
  • CSRF / SSRF 攻击与防御
  • 文件上传漏洞 & 文件包含 (LFI/RFI)
  • PHP / Java 常见反序列化漏洞

🔐 密码学 (Crypto) & 逆向 (Reverse)

从数学之美到到底层逻辑的硬核挑战。

核心知识点：

  • 古典密码体制与统计分析
  • RSA、AES 等现代密码学体系破解
  • IDA Pro / OllyDbg 静态分析与动态调试
  • 汇编语言 (x86/ARM) 逆向工程基础

🛡️ Pwn (二进制安全) & Misc (杂项)

系统级漏洞利用与脑洞大开的信息隐藏艺术。

核心知识点：

  • 缓冲区溢出 (Buffer Overflow) 原理
  • ROP 链构造与 Shellcode 编写
  • 流量分析 (Wireshark) 与隐写术 (Steganography)
  • 内存取证与各种奇葩文件格式解析
